# 🧮 Calculadora de Interés Compuesto

![Java](https://img.shields.io/badge/Java-17%2B-blue)
![JavaFX](https://img.shields.io/badge/JavaFX-21-orange)
![Maven](https://img.shields.io/badge/Maven-3.8.1-red)
[![License](https://img.shields.io/badge/License-MIT-green.svg)](https://opensource.org/licenses/MIT)

Aplicación JavaFX para calcular el crecimiento de inversiones con interés compuesto, incluyendo aportaciones periódicas y ajuste por inflación.


## ✨ Características

- Cálculo de interés compuesto con/sin aportaciones periódicas
- Soporte para frecuencias de aportación: mensual, trimestral, semestral o anual
- Ajuste opcional por inflación estimada
- Interfaz intuitiva con validación de datos
- Exportación de resultados (próximamente)

## 🚀 Cómo ejecutar

### Requisitos
- JDK 17 o superior
- Maven 3.8+

### Pasos
```bash
# Clonar el repositorio
git clone https://github.com/jshevvik/calculadora-intereses.git

# Navegar al proyecto
cd calculadora-intereses

# Compilar y ejecutar
mvn clean javafx:run
